O que é UX Design?


Por muito tempo, a ideia de design foi associada ao design gráfico, que, pode-se dizer, é basicamente a aparência de um produto. À medida que a tecnologia digital e nossas expectativas sobre as interações digitais aumentaram, começamos a nos concentrar cada vez mais na parte sentir de um design, também conhecida como a experiência do usuário. Se UX é a experiência que um usuário tem ao interagir com um produto, então UX Design é o processo pelo qual um designer tentou determinar o que essa experiência será. É importante salientar que não se pode projetar experiências como uma entidade formal, é possível apenas projetar as condições para se chegar a uma experiência pretendida. O papel do designer de experiência de uso está diretamente envolvido no processo de tornar um produto útil, utilizável e agradável para seus usuários mas muitas das vezes este é utilziado para prender o usuário, aplicativos como Instagram, facebook, Gmail... são exemplos de designes criados para atrair sua atenção o maximo possivel, esse assunto é muito bem abordado em um documentario da Netflix "O Dilema das Redes", recomendo de mais caso a professora não tenha assitido.
